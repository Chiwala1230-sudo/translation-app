# Translator App

A web app that translates text between English and French. Built for a full stack development assignment.

## What it does

Type or paste text, pick a language, and get a translation. Opens with "Hello, how are you" in English, translates to French.

## Features

- 500 character limit with counter
- Swap languages with one click
- Listen to text (speaker button)
- Copy text to clipboard
- Auto-translates while typing
- Works on mobile and desktop

## Note about "Detect Language"

The assignment asked for a "Detect Language" option but the free MyMemory API doesn't support it. If selected, the app defaults to English. A paid API like Google Translate would fix this.

## How to run

1. Download all files
2. Put them in one folder
3. Double-click index.html
4. No server needed
